## ğŸš€ Quick Start

### Local Development

The website is a static HTML site that can be run with any web server:

```bash
cd landing-site
python -m http.server 8080
```

Or using npm:

```bash
cd landing-site
npm start
```

Then open your browser to `http://localhost:8080`

### Deployment

The website can be deployed to any static hosting service:

- **GitHub Pages**: Simply push to your repository and enable Pages
- **Netlify**: Drag and drop the `landing-site` folder
- **Vercel**: Connect your repository and deploy
- **Any web server**: Upload the `landing-site` folder contents
